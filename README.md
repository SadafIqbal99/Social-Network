SocialNet - Social Media Platform Demo
A fully functional social media platform demo built with HTML, CSS, and JavaScript. No backend required - runs entirely in the browser with simulated real-time features!

**🚀 Features
📱 Core Features**
News Feed - Scroll through posts with infinite scroll

User Profiles - Complete profile pages with about section, posts, and stats

Create Posts - Share text, images, and media with privacy settings

Like & Comment - Interact with posts through likes and comments

Friend System - Send/accept friend requests, manage friends list

Real-time Chat - Messaging system with conversations

Notifications - Live notification center with badges

Stories - Visual stories section at the top of feed

**🎨 UI/UX Features**
Responsive Design - Works perfectly on desktop, tablet, and mobile

Dark/Light Mode - Smooth theme switching

Animations - Smooth transitions and hover effects

Modals - Beautiful popup modals for posts and media

Loading States - Spinners and skeleton loading

Toast Notifications - Real-time alerts

**🔥 Advanced Features**
Real-time Simulation - Live updates, online status, typing indicators

Media Viewer - Click images to view fullscreen

Search - Search for friends and content

Trending Topics - What's popular right now

Friend Suggestions - "People you may know" recommendations

Online Friends - See who's online in real-time

Story Creation - Add to your story (UI ready)

**🛠️ Technologies Used**
HTML5 - Structure and semantics

CSS3 - Styling, animations, responsive design

JavaScript (ES6+) - All functionality and interactions

Font Awesome - Icons and UI elements

Unsplash - Demo images and placeholders
